# How to:

### _install_

For _yarn_:

```sh
yarn install
```

For _npm_:

```sh
npm install
```

### _start_

For _yarn_:

```sh
front  - yarn start
server - cd server/ node server.js
```

For _npm_:

```sh
front  - npm run start
server - cd server/ node server.js
```
